# School Management System

## Overview

The School Management System is a full-stack web application developed to manage school operations efficiently. It provides separate modules for administrators, teachers, students, and parents.

## Technologies Used

### Frontend

* ReactJS
* HTML5
* CSS3
* JavaScript
* Vite

### Backend

* Java
* Spring Boot
* Spring Security
* REST API

### Database

* MySQL

## Features

### Admin Module

* Dashboard
* Student Management
* Teacher Management
* Parent Management
* Timetable Management
* School Operations Management

### Student Module

* Student Registration
* Login Authentication
* View Profile
* View Timetable

### Teacher Module

* Login Authentication
* View Assigned Information
* Manage Student Data

### Parent Module

* View Student Information
* Monitor Academic Activities

### Security Features

* Authentication and Authorization
* Secure Login System
* Role-Based Access Control
